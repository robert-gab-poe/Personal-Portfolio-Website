# Personal-Portfolio-Website

## Robert Gabriel - Space Portfolio

A visually stunning portfolio website with a cosmic space theme featuring:

- **Animated Starfield Background** - Dynamic stars, nebula effects, and shooting stars
- **Hero Section** - Introduction with floating avatar and glow effects
- **Skills Constellation** - Technical skills organized into categories
- **Programming Languages** - Progress bars showing proficiency levels
- **Projects Galaxy** - GitHub repositories fetched dynamically
- **Links Section** - Connect via GitHub, LinkedIn, and Instagram
- **Contact Form** - Send messages across the cosmos

### Tech Stack

- React + Vite
- Framer Motion (animations)
- CSS3 with space theme

### Run Locally

```bash
npm install
npm run dev
```
